# SPARK Architecture Documentation

This directory documents the architectural design of SPARK.

- `workspace-platform.md` — Platform-level artifact lifecycle and traceability
- `workspace-chat-notebook.md` — Chat-first analytical notebook UI
- `ai-augmentation.md` — AI usage model and constraints
- `security-model.md` — Trust boundaries and threat model
- `mcp-integration.md` — External context integration design

These documents describe design intent and constraints, not delivery commitments.
